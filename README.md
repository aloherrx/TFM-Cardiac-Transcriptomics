## Overview

This repository contains the R code used for all bioinformatic analyses in the Master's thesis. The study profiles the cardiac transcriptome of diet-induced obese C57BL/6J mice subjected to a 12-week dietary intervention with refined olive oil (ROO) or refined olive oil enriched with omega-3 fatty acids (ROO+ω-3), compared to a control diet.

Analyses include differential expression (DESeq2), functional enrichment (GSEA against Hallmark, GO and KEGG gene sets), and a range of visualisations including volcano plots, heatmaps, chord diagrams and enrichment score plots.

## How to run

1. Clone this repository:
```bash
git clone https://github.com/aloherrx/TFM-Cardiac-Transcriptomics
cd TFM-Cardiac-Transcriptomics
```

2. Place the count matrix in the `input_data/` folder:
```bash
mkdir input_data
cp /path/to/gene_count_matrix.csv input_data/
```

3. Open `script_tfm_v3.Rmd` in RStudio and knit to HTML, or run from the command line:
```r
rmarkdown::render("script_tfm_v3.Rmd")
```

All outputs will be written to `results_mouse_heart_diet/` organised into six subdirectories.

## Key analysis parameters

| Parameter | Value |
|-----------|-------|
| Minimum count filter | >= 10 counts in >= 3 samples |
| Adjusted p-value cutoff | < 0.05 (Benjamini-Hochberg) |
| log2 fold change cutoff | >= 1 |
| Reference genome | GRCm39 (GCA_000001635.9) |
| Aligner | STAR v2.7.11b |
| Quantification | StringTie v2.1.3b |
